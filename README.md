## 4. Giới thiệu đồ án

### Tên đồ án
**Phụng Lộc Coffee**  

### Mô tả
Đây là đồ án nhóm được thực hiện במסגרת môn **IS208 - Quản lý dự án công nghệ thông tin**.  
Dự án hướng đến việc xây dựng **hệ thống quản lý chuỗi cửa hàng cà phê Phụng Lộc Coffee**, hỗ trợ quản lý hoạt động kinh doanh tại nhiều chi nhánh một cách hiệu quả và đồng bộ.

### Mục tiêu chính
- Quản lý thông tin chi nhánh
- Quản lý nhân viên và tài khoản
- Quản lý nguyên vật liệu, nhập kho, xuất kho
- Theo dõi doanh thu và báo cáo
- Đồng bộ dữ liệu giữa các chi nhánh
- Hỗ trợ vận hành hệ thống tập trung

---

## Công nghệ dự kiến sử dụng
- **Ngôn ngữ lập trình:** C# / Java / SQL Server *(tùy theo định hướng nhóm)*
- **Mô hình phát triển:** Desktop Application
- **Cơ sở dữ liệu:** Tập trung, hỗ trợ quản lý đa chi nhánh
